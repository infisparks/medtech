# MedTechMakeIt

A professional educational platform for medical students, featuring a public landing page, inquiry and registration forms, a student mobile app, and an admin dashboard.

## Features

- **Landing Page**: Modern, responsive design with glassmorphism effects.
- **Inquiry Form**: Professional contact form.
- **Class Registration**: Detailed registration flow for new students.
- **Student App**: Mobile-first experience with bottom navigation, course tracking, and schedule.
- **Admin Dashboard**: Comprehensive management interface with sidebar navigation and stats.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
- **Animations**: Framer Motion (ready to use)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Routes

- `/`: Landing Page
- `/inquiry`: Inquiry Form
- `/register`: Class Registration
- `/studentapp`: Student Mobile App Dashboard
- `/adminapp`: Admin Dashboard
